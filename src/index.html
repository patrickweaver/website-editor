📟 One Pager

About

One Pager is a website that can edit itself. It is a single index.html file with some JavaScript that, when accessed offline, allow editing the HTML file and saving your changes.

If you see the “Local Controls” area below then you are either editing locally or viewing the demo page at doodles.patrickweaver.net/one-pager.

How do I try it?

You just need a copy of this HTML file to get started. You can either do File > Save Page As, or clicking the “Save All Changes to Local File” button at the very bottom of the Local Controls area.

Once you have the file on your computer, open it in a browser (I have only tested in Chromium based browsers so far) using File > Open File. You can click on any Heading, Paragraph or Image to edit their content or formatting, or edit page-level properties in the Local Controls area.

What can I put on a One Pager website?

One Pager currently supports: Headings, Paragraphs and Images. The code is open source and there are issues on GitHub for other features if you want to contribute.
